# 回測策略說明文件 (Backtest Strategies Comparison)

本文件詳細說明 `backtest_trand.py` 中實作的四種交易策略及其背後的會計邏輯。

## 策略定義

| 策略名稱 | 邏輯核心 | 特點與適用場景 |
| :--- | :--- | :--- |
| **Strategy A (Hold)** | **長期持有 (Buy & Hold)** | 基準線。初始買入後一路持有到期末。適用於強勢牛市。 |
| **Strategy B (Trend)** | **動態高低點 (Dynamic Trend)** | **核心策略。** 監控區間最高/最低點。回落 X% 賣出，回升 Y% 買入。適合捕捉大波段、過濾小震盪。 |
| **Strategy C (SMA)** | **均線交叉 (SMA20 + Cool-down)** | 價格站上/跌破 20 日均線立即交易，但隨後進入 **3 天冷卻期**。適合趨勢明確的市場，但在盤整期易被通殺。 |
| **Strategy D (ATR)** | **自適應波動 (ATR Adaptive)** | 門檻不再固定，而是根據最近 14 天的 **ATR (波動率)** 自動調整。波動大時門檻自動推開，波動小時自動縮緊。 |

## 會計與損耗邏輯 (Realistic Accounting)

為了模擬真實交易中的「折損」，本回測系統採用了以下嚴格的會計準則：

1. **交易稅與手續費 (Fees & Taxes)**:
   - **台股**: 手續費 0.1425% (預設 65 折) + 賣出交易稅 0.3%。
   - **美股**: 手續費 0.5% (複委託標準) + 0% 稅金。
2. **滑價損耗 (Slippage)**:
   - 每次買賣交易額外扣除 **0.1%**。模擬實際成交時無法精準買在收盤價的價差。
3. **期末強制清算 (Final Liquidation)**:
   - 回測結束當天，如果手中仍持有股票，系統會強制按當天價格「虛擬賣出」，並扣除手續費、稅金與滑價，得出最終的**淨現金資產**。
4. **複利計算**:
   - 所有的獲利都會重新投入下一次交易，虧損亦然。

## 實戰結論 (Conclusion)

- **牛市 (如 2025 年)**: **Strategy A (Hold)** 往往表現最穩、成本最低。策略 B/D 需要付出「保險費 (錯失漲幅)」。
- **熊市 (如 2022 年)**: **Strategy B/D** 展現強大的逃頂保命能力，能將 -30% 的巨虧收斂甚至轉正。
- **黃金參數**: 根據大量測試，**「賣出門檻 9%」** 是一個極具韌性的穩健參數，能平衡多數股票的雜訊與獲利空間。


## 實戰應用（以台積電為例）

要將這一套回測出來的「Strategy B (動態趨勢)」應用在 2330 (台積電) 上，其實非常直覺。用白話來說，這就是一套**「跟隨大行情，無視小震盪」**的懶人避險法。

以下是實際操作的三個步驟：

第一步：設定「黃金盔甲」 (修改 
stock_list.txt
)
根據回測，台積電最穩健的門檻大約是 (回升 9%, 回落 9%)。 您在 
stock_list.txt
 裡應該這樣寫：

台積電, 2330, 9, 9

這代表：

賣出防線：股價從最高點掉下來 9% 我就撤退（保住股利、保住本金）。
買入進攻：股價從最低點彈上來 9% 我就進場（確認趨勢翻轉）。
第二步：讓系統每天幫您「抓體溫」
您不需要每天盯盤看有沒有跌破 9%。您只需要依賴 
stock_monitor.py
：

系統會自動紀錄：它會幫您記得台積電最近一個月的「最高點」與「最低點」。
每天通報：它會告訴您：「台積電現在離高點跌了 5%，還沒到 9%，繼續抱著。」
達標警示：一旦哪天收盤真的跌了超過 9%，它會傳送訊息給您說「⚠️跌幅達標」，這就是這套策略通知您該「停利/停損」的時候了。
第三步：實戰中的心理與動作 (白話腳本)
這套方法在 2330 上的實際體感會是這樣：

多頭大漲時 (如 2025)： 台積電從 800 漲到 1000，中間偶爾回檔 3-5%。因為我們設了 9% 的寬容空間，您的通知會一直叫您「抱住」，您就不會在半路被洗掉，能一路吃到整個波段。
空頭大跌時 (如 2022)： 台積電從 688 開始崩盤，跌到 620 元左右的時候 (約跌 10%)。這套系統會發出警報，叫您在 620 元左右清空。
雖然您會覺得「賣在 620 比 688 少賺了」，但回頭看 2022 年台積電一路跌到 370 元。
您的 9% 策略幫您擋掉了後面那一段 -40% 的歷史慘劇。
低檔反彈時： 當台積電從 370 元低點彈回到 404 元時 (回升約 9%)，系統會叫您買進。您雖然沒買在最低的 370，但您買在了「確定的起漲點」。
