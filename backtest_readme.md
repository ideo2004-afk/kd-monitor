
# 股市投資策略回測工具 (Stock Backtesting Tool)

這是一個強化的股市回測工具，支援台股與美股，並具備「雙門檻」趨勢策略優化功能。

## 核心功能

*   **雙門檻策略 (Strategy B)**：可分別設定「買入門檻」與「賣出門檻」，模擬更精細的進出節奏。
*   **2D 參數優化**：自動搜尋 3% 到 15% 之間最佳的買賣門檻組合（財富密碼）。
*   **自動市場判斷**：
    *   **台股 (數字代號)**：自動計算 65 折手續費與 0.3% 證交稅。
    *   **美股 (英文代號)**：自動計算 0.5% 複委託手續費，免證交稅。
*   **長期持有基準 (Strategy A)**：自動對比「買入持有」的收益，作為評估標準。

## 安裝與設定

1.  **建立虛擬環境**:
    ```bash
    python3 -m venv venv
    ```

2.  **安裝依賴套件**:
    ```bash
    ./venv/bin/pip install yfinance pandas matplotlib
    ```

### 3. (選配) 開啟 Gemini AI 自動化分析
本工具支援呼叫 Gemini 2.0 Pro 進行量化分析：
*   複製 `.env.example` 為 `.env`:
    ```bash
    cp .env.example .env
    ```
*   在 `.env` 中填入您的 `GEMINI_API_KEY` ([由此取得](https://aistudio.google.com/app/apikey))。
*   開啟後，在 `--optimize` 模式結束時，系統會自動產出一段 AI 量化建議報告。

| 參數 | 說明 | 預設值 |

## 策略 B (趨勢追蹤) 深度說明

策略 B 是一種「順勢而為」的策略，其核心在於**不預測底部與頭部**，而是等待趨勢確認後才行動。

### 1. 執行邏輯與訊號
程式會持續追蹤股價的「波段高點 (Peak)」與「波段低點 (Valley)」：

*   **買入訊號 (Buy Signal)**：
    *   當您手上有現金時，程式會紀錄這段時間出現過的最低價（谷底）。
    *   一旦股價從谷底反彈超過 **`buy_t` (買入門檻)**，程式判定「漲勢啟動」，立即全倉買入。
    *   *公式：當前價格 >= 谷底最低價 * (1 + buy_t)*

*   **賣出訊號 (Sell Signal)**：
    *   當您手上持有股票時，程式會紀錄買入後出現過的最高價（高峰）。
    *   一旦股價從高峰回落超過 **`sell_t` (賣出門檻)**，程式判定「趨勢轉弱」，立即全倉賣出避險。
    *   *公式：當前價格 <= 高峰最高價 * (1 - sell_t)*

### 2. 為什麼要使用「雙門檻」？
*   **非對稱性**：不同的股票性格不同。例如台積電適合「寬買/寬賣」來避開震盪；而投機股可能適合「寬買/窄賣」來快速保命。
*   **過濾雜訊**：門檻設得越寬，交易次數越少，能避開所謂的「假突破」與「假摔」，但進場點會稍晚。

### 3. 此策略的優缺點
*   **優點**：能避開像 2022 年那種毀滅性的大崩盤（因為在跌 10%-15% 時就會強制離場），並在市場重新走強時自動接回。
*   **缺點**：在「橫盤震盪（箱型整理）」的行情下，可能會出現多次追高殺低，導致手續費損失。
